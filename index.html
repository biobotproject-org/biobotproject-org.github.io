Built for the field

🔥

Early Wildfire Detection

BMV080 particulate and BME688 gas sensors detect smoke signatures before fires spread, enabling faster emergency response.

📡

Cellular Connectivity

Blues Notecard provides reliable LTE-M connectivity from remote deployments, no Wi-Fi required.

⚡

Ultra-Low Power

Engineered for multi-week battery life in the field. Deploy anywhere, forget about charging.

🔧

Flexible Hardware

Modular design adapts to your deployment. Swap sensors, swap radios, or extend the platform for new use cases without starting from scratch.

📊

Live Dashboard

Real-time sensor data, deployment maps, and alert management from any browser.

🔓

Fully Open Source

Hardware schematics, firmware, and cloud stack are all open source under GPLv3.
